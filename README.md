# 📊 Sales Data Analysis Dashboard

## 📌 Project Description
This project analyzes sales and customer data to identify trends, improve business performance, and support data-driven decision making.

## 🛠 Tools & Technologies
- Excel
- Power BI
- SQL

## 📂 Dataset
The dataset contains:
- Order ID
- Date
- Customer Name
- Region
- Category
- Product
- Sales
- Profit
- Quantity
- Order Status

## 📈 Key Insights
- Identified top-performing customers and products
- Analyzed monthly sales trends
- Compared profit across categories
- Found regional sales distribution patterns

## 📊 Dashboard Features
- KPI Cards (Total Sales, Profit, Orders)
- Sales Trend Line Chart
- Top Customers Bar Chart
- Category Performance
- Regional Map Visualization
